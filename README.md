# N-Gram Predictor

This project implements an N-Gram model for text prediction.

## Project Structure

- `config/`: Configuration files
- `data/`: Data files
- `src/`: Source code
- `tests/`: Test files
- `main.py`: Entry point

## Installation

1. Install dependencies: `pip install -r requirements.txt`

## Usage

Run `python main.py` to start the application.
